Baby Cry Monitor 개인정보 처리방침

시행일: 2026년 5월 22일

Baby Cry Monitor는 아기 울음 감지와 보호자 알림 기능을 제공하기 위해 필요한 범위에서만 정보를 처리합니다. 이 문서는 Baby Cry Monitor 유료 버전과 체험판 버전에 공통으로 적용됩니다.

1. 처리하는 정보

앱은 다음 정보를 기능 제공을 위해 처리할 수 있습니다.

  • 마이크 입력: 아기 울음 또는 주변 소리 감지를 위해 기기 안에서 실시간으로 사용됩니다.
  • 보호자 전화번호: 울음 감지 시 전화 알림을 보내기 위해 사용자가 입력한 번호를 기기에 저장합니다.
  • 아기 이름 또는 별명: 앱 화면에 표시하기 위해 사용자가 입력한 값을 기기에 저장합니다.
  • 아기 사진: 사용자가 선택한 사진을 앱 화면 표시용으로 기기에 저장합니다.
  • 연락처 정보: 사용자가 연락처에서 전화번호를 선택할 때 선택한 번호를 불러오기 위해 사용됩니다.
  • 기기 상태 정보: 원격 상태 확인 기능을 켠 경우 모니터링 상태, 마지막 갱신 시각, 배터리 상태, 앱 버전, 기기 모델명이 전송될 수 있습니다.
  • 오류 정보: 앱 안정성 개선을 위해 Firebase Crashlytics를 통해 충돌 로그와 기기 환경 정보가 수집될 수 있습니다.

2. 정보의 이용 목적

  • 아기 울음 감지와 알림 기능 제공.
  • 보호자 전화 알림 기능 제공.
  • 아기 이름, 사진, 테마 등 사용자 설정 유지.
  • 원격 상태 확인 링크를 통한 모니터링 상태 표시.
  • 앱 오류 분석, 품질 개선, 안정성 향상.

3. 마이크와 오디오 처리

앱은 울음 감지를 위해 마이크 권한을 사용합니다. 마이크 입력은 감지 기능을 위해 기기에서 처리되며, 통화나 원격 상태 확인 기능을 위해 음성 녹음 파일을 서버로 업로드하지 않습니다.

4. 원격 상태 확인 기능

사용자가 원격 상태 확인 기능을 켠 경우, 보호자가 QR 코드 또는 링크를 통해 현재 모니터링 상태를 확인할 수 있습니다. 이 기능을 위해 Firebase Realtime Database에 다음 정보가 저장될 수 있습니다.

  • 모니터링 중 또는 정지 상태.
  • 마지막 상태 갱신 시각과 모니터링 시작 시각.
  • 배터리 잔량과 충전 여부.
  • 앱 버전과 기기 모델명.

상태 확인 링크에는 무작위로 생성된 식별자가 포함됩니다. 링크를 받은 사람은 해당 상태 페이지를 볼 수 있으므로, 사용자는 링크를 신뢰할 수 있는 보호자에게만 공유해야 합니다.

5. 제3자 서비스

앱은 기능 제공과 안정성 개선을 위해 다음 서비스를 사용할 수 있습니다.

  • Firebase Realtime Database: 원격 상태 확인 기능 제공.
  • Firebase Crashlytics: 앱 충돌과 오류 분석.
  • Google Play 서비스: 앱 배포, 업데이트, 기기 환경 지원.

각 서비스에서 처리하는 정보는 해당 서비스 제공자의 정책에 따라 관리될 수 있습니다.

6. 정보 보호 방법

앱은 개인정보를 필요한 기능 제공 범위 안에서만 처리하도록 설계되어 있습니다. 원격 상태 확인 정보는 무작위 식별자가 포함된 링크를 통해 접근되며, 사용자는 이 링크를 신뢰할 수 있는 사람에게만 공유해야 합니다. 외부 서비스로 전송되는 정보는 Firebase 등 서비스 제공자의 보안 체계에 따라 전송 및 관리됩니다.

7. 정보의 보관과 삭제

  • 전화번호, 아기 이름, 설정값, 사진은 사용자의 기기에 저장되며 앱 삭제 시 함께 삭제됩니다.
  • 원격 상태 확인 정보는 기능 제공을 위해 서버에 저장될 수 있으며, 더 이상 필요하지 않은 데이터는 서비스 운영 과정에서 삭제될 수 있습니다.
  • Crashlytics 오류 로그는 앱 안정성 분석을 위해 제한된 기간 동안 보관될 수 있습니다.
  • 원격 상태 확인 정보 삭제 요청은 아래 문의 이메일로 보낼 수 있습니다.

8. 권한 사용 안내

  • 마이크: 아기 울음 감지.
  • 전화: 울음 감지 시 보호자에게 전화 걸기.
  • 카메라와 사진 접근: 아기 사진 선택 및 등록.
  • 알림과 포그라운드 서비스: 모니터링 상태 유지 및 표시.
  • 배터리 최적화 예외: 화면이 꺼져도 감지 기능이 안정적으로 작동하도록 지원.
  • 인터넷: 원격 상태 확인과 오류 분석 기능 제공.

9. 아동 개인정보

앱은 보호자가 직접 입력하거나 선택한 정보 외에 아동의 개인정보를 별도로 요구하지 않습니다. 아기 이름이나 사진 입력은 선택 사항이며, 사용자는 입력하지 않고도 주요 기능을 사용할 수 있습니다.

10. 이용자의 선택

  • 원격 상태 확인 기능은 설정에서 켜거나 끌 수 있습니다.
  • 전화번호, 아기 이름, 사진은 앱 설정 또는 앱 데이터 삭제를 통해 제거할 수 있습니다.
  • 권한은 Android 시스템 설정에서 언제든지 변경할 수 있습니다. 다만 일부 권한을 거부하면 관련 기능이 제한될 수 있습니다.

11. 개인정보 처리방침 변경

앱 기능, 관련 법령, 서비스 운영 방식이 변경되는 경우 이 개인정보 처리방침이 업데이트될 수 있습니다. 변경된 내용은 이 페이지에 게시됩니다.
